## 🚀 Featured Projects

### 🏥 Offline-First Healthcare App
> *"The internet is a privilege. Healthcare is not."*

Built for **ASHA workers in rural India** — this Flutter + Spring Boot app works completely offline, stores patient data in SQLite, and syncs automatically when connectivity returns.

<p>
  <img src="https://img.shields.io/badge/Flutter-02569B?style=flat-square&logo=flutter&logoColor=white"/>
  <img src="https://img.shields.io/badge/Spring_Boot-6DB33F?style=flat-square&logo=springboot&logoColor=white"/>
  <img src="https://img.shields.io/badge/JWT-000000?style=flat-square&logo=jsonwebtokens&logoColor=white"/>
  <img src="https://img.shields.io/badge/SQLite-07405E?style=flat-square&logo=sqlite&logoColor=white"/>
</p>

<details>
<summary>📂 <b>View features</b></summary>
<br/>

| Feature | Details |
|---------|---------|
| 🔐 **Auth** | Stateless JWT login — no session, no risk |
| 📋 **Records** | Patient CRUD with local-first persistence |
| 📴 **Offline Mode** | Full app functionality with zero internet |
| 🔄 **Auto Sync** | Pushes local data to server on reconnect |
| 🖼️ **Media** | Image capture with local cache |
| 🛡️ **Security** | Spring Security-hardened REST endpoints |

</details>

### 🤖 AutoAuditor — AI Expense Auditor Agent
> *"Corporate expenses audited in seconds, not weeks."*

A full-stack AI application that automates corporate expense auditing — detecting fraud, flagging policy violations, and delivering real-time analytics through an intelligent multi-phase pipeline. Upload a CSV, get a complete audit report with AI-generated insights.

<p>
  <img src="https://img.shields.io/badge/Flutter-02569B?style=flat-square&logo=flutter&logoColor=white"/>
  <img src="https://img.shields.io/badge/FastAPI-009688?style=flat-square&logo=fastapi&logoColor=white"/>
  <img src="https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white"/>
  <img src="https://img.shields.io/badge/Llama_3-7B3F00?style=flat-square&logo=meta&logoColor=white"/>
  <img src="https://img.shields.io/badge/Pandas-150458?style=flat-square&logo=pandas&logoColor=white"/>
</p>

<details>
<summary>📂 <b>View features</b></summary>
<br/>

| Phase | What happens |
|-------|-------------|
| 📥 **Data Prep** | Dynamic schema detection, cleaning, behavioural tagging |
| 📏 **Rule Engine** | Budget threshold · duplicate · frequency · missing-doc violations |
| 🧠 **AI Audit** | Statistical anomaly detection, risk scoring, fraud signal correlation |
| 📊 **Dashboard** | Executive summary, spending trends, top spenders, category breakdown |
| 💬 **LLM Insights** | Llama 3 generates natural language summaries & cost optimisation tips |

**Architecture:** `Flutter UI → FastAPI → 4-phase pipeline → JSON dashboard`

</details>
