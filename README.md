## 🚀 Featured Projects

### 🏥 Offline-First Healthcare App
> *"The internet is a privilege. Healthcare is not."*

Built for **ASHA workers in rural India** — this Flutter + Spring Boot app works completely offline, stores patient data in SQLite, and syncs automatically when connectivity returns.

<p>
  <img src="https://img.shields.io/badge/Flutter-02569B?style=flat-square&logo=flutter&logoColor=white"/>
  <img src="https://img.shields.io/badge/Spring_Boot-6DB33F?style=flat-square&logo=springboot&logoColor=white"/>
  <img src="https://img.shields.io/badge/JWT-000000?style=flat-square&logo=jsonwebtokens&logoColor=white"/>
  <img src="https://img.shields.io/badge/SQLite-07405E?style=flat-square&logo=sqlite&logoColor=white"/>
</p>

<details>
<summary>📂 <b>View features</b></summary>
<br/>

| Feature | Details |
|---------|---------|
| 🔐 **Auth** | Stateless JWT login — no session, no risk |
| 📋 **Records** | Patient CRUD with local-first persistence |
| 📴 **Offline Mode** | Full app functionality with zero internet |
| 🔄 **Auto Sync** | Pushes local data to server on reconnect |
| 🖼️ **Media** | Image capture with local cache |
| 🛡️ **Security** | Spring Security-hardened REST endpoints |

</details>
