# 🚀 Projects

## 🏥 Offline-First Healthcare App

A **Flutter + Spring Boot healthcare application** designed for **ASHA workers in rural areas** where internet connectivity is unstable.  
The system follows an **offline-first architecture**, allowing workers to store patient data locally and sync automatically when internet connectivity is restored.

### Tech Stack
- Flutter (Dart)
- Spring Boot
- Spring Security
- JWT Authentication
- SQLite (Offline Database)
- REST APIs

### Key Features
- Secure **JWT-based login and registration**
- **Patient record creation and management**
- **Task management system for ASHA workers**
- **Offline data storage using SQLite**
- **Automatic background data synchronization**
- **Image handling with local caching**
- **Secure backend APIs using Spring Security**

### Concepts Implemented
- Stateless authentication with JWT
- Offline-first mobile architecture
- Local database + server synchronization
- REST API security
- Handling unreliable internet connectivity
